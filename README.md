# MERN Todo App
MERN App:  MongoDB, Express, ReactJS, Node.js and GraphQL

## 1. Install Server
```
cd server
```
```
npm install
```

## 2. Instal App
```
cd client
```
```
npm install
```

## 3. Start Server
At server folder
```
node index.js
```

## 4. Run App
Go to client folder and have a look on the readme.md
